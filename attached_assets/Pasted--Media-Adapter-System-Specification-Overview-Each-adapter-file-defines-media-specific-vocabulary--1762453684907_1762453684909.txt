Overview

Each adapter file defines media-specific vocabulary, constraints, and emphasis that modify or extend a core system prompt.
Adapters allow your prompt builder to deliver precise creative guidance per image style without duplicating the entire instruction set.

Core idea:

Final Prompt = Core Prompt + Selected Media Adapter
